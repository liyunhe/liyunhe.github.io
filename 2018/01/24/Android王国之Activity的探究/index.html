发表于 2018-01-24 | 分类于 Android学习 |

什么是活动

活动是包含所用用户界面的组件，主要用于和用户进行交互，一个应用包含很多个活动。

活动的基本用法

创建一个活动：在你的主包下 ，New - Acticity - Empty Acticity AndroidStudio会显示出configgure Acticity 你可以设置他的 name 和layout name 点击OK 创建成功

代码

protected void onCreate(Bundle savedInstanceState) {
super.onCreate(savedInstanceState);
