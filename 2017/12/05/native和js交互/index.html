JavaScript API介绍

dsBridge

“dsBridge”是一个内置对象，它有两个方法“调用”和“注册”

dsBridge.call(method,[args,callback])

同步和异步调用Object-C api。

方法：Object-c方法名称

参数与json对象

回调（String returnValue）：回调来处理结果。只需要异步调用。

dsBridge.register(methodName,function)

为OC调用注册javascript方法。

methodName：javascript函数名称

函数：javascript方法体。

Object-c API signature

更多关于DWebview

有三个WebView可用，DWKwebview，DUIwebview和DWebview，他们都提供相同的接口，你可以用户任何你想要的。值得一提的是，DWebview只是DWKwebview和DUIwebview的代理，而ios系统的vesion> = 8.0，DWKwebview将被使用，否则，DUIwebview将会是。

警告

如果您使用的是DUIwebview，请不要设置代理prop。因为代表道具已经内置，请设置WebEventDelegate！
